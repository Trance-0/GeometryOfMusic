# 0.0.6

**Date:** 2026-04-20.

Layout pass. Three small-but-visible UX changes.

## Mobile-friendly navbar

On narrow viewports (≤ 720 px) the musical-spaces list now collapses
into a hamburger dropdown on the right side of the topbar.

- New `#nav-toggle` button (three-line icon) sits between the brand
  and the `?` help button on mobile; it is hidden on wider screens.
- Tapping it toggles `.open` on `#space-nav`; the nav becomes a
  full-width dropdown under the topbar with each space as a
  stacked, full-width button.
- Clicking any space selects it and auto-closes the dropdown, as
  does tapping anywhere outside the nav.
- The `.brand-title` sub-label is hidden under 720 px to make room;
  the `GoM` mark stays visible.

## Mouse-button swap: right-drag rotates, middle-drag pans

Many users expect right-click to rotate a 3D view (Blender-style).
Swapped:

- **Right drag** → rotate (was middle).
- **Middle drag** → pan (was right).
- **Left click** still picks a node; **scroll** still zooms.

`OrbitControls.mouseButtons.RIGHT` is now `THREE.MOUSE.ROTATE`, and
`MIDDLE` is `THREE.MOUSE.PAN`. The context-menu suppressor stays on
the canvas so right-drag doesn't pop the browser menu. There is no
translation drag on right click anymore — pan is only on middle.

Tour modal copy, the transport hint text, and
[`docs/usage.md`](../usage.md) updated in the same round so the
documentation agrees with the actual mapping.

## Reset view + Hide torus moved to the top-right

Previously **Reset view** was top-left and **Hide torus** next to it;
the interval-class legend lived on the top-right. The two controls
are now together on the **top-right**, on one horizontal row, with
the legend stacked directly beneath them. The top-left overlay
container was removed.

- `#lattice-overlay-tl` → `#lattice-overlay-tr` (flex column,
  aligned to the right edge).
- New inner `.lattice-controls` flex row holds the button and the
  checkbox.
- `.legend` absolute positioning stripped — it is now a regular
  flow child of `.lattice-overlay-tr`.
- On narrow viewports the overlay shifts to `top: 6px; right: 6px`,
  the controls wrap if needed, and the legend caps at
  `calc(100vw - 24px)`.
