# 0.0.3

**Date:** 2026-04-20.
**Commit:** `1460b1e`.

Musical expressivity round: per-track curves on the torus, chord
durations with drag-to-extend, and familiar starter progressions.
Follow-up pass on the light-theme wireframe.

## Per-track curves

- `TorusView.setProgressionPath` replaced with `setTrackPaths`, which
  takes one `{ color, dyads }` entry per track and draws a
  **Catmull-Rom (centripetal) curve** in that color.
- Three curves in three colors: **gold** (Lead), **sky blue** (Bass),
  **pink** (Pad).
- `TrackRow.color` added; each track header in the timeline now
  shows a colored swatch that matches its curve on the torus.

## Chord durations + drag-to-extend

- Timeline data model upgraded from per-cell `Dyad | null` to
  per-cell `Placement | null` where `Placement = { dyad, duration }`.
- A filled chord renders as **one grid element spanning `duration`
  columns** with a drag grip on its right edge.
- Drag the grip to extend or shrink. Snaps to cell boundaries; hard
  stops at the next chord on the same track (never overlaps).
- Scheduler multiplies per-step duration by `placement.duration`, so
  a held chord plays as **one long note**, not repeated attacks.

## Starter templates

- New `src/presets.ts` with four progressions voiced across three
  tracks at natural chord durations:
  - **I–V–vi–IV (C major)** — the "axis" pop progression
    (*Let It Be*, *Don't Stop Believin'*, etc.).
  - **Pachelbel's Canon (D major)** — the eight-chord ground bass
    compressed to four bars.
  - **12-bar blues (C)** — canonical I-IV-V blues with dom7
    voicings on the IV and V chords and a V7 turnaround.
  - **ii–V–I (C major)** — jazz turnaround Dm7 → G7 → Cmaj7 with
    the resolution held over two bars.
- Preset dropdown in the first transport row loads a preset (clears
  the grid, sets meter / bars / subdivision, writes placements).
- I–V–vi–IV loads on app start.

## Wireframe cleanup

- `TorusGeometry(R, r, 48, 96)` → `(R, r, 8, 24)`. The wireframe is
  now a readable low-poly grid instead of dense visual noise.
- `applySceneTheme` retints the shell **and** the voice-leading edges
  for the active theme (subtle grey in light mode, close to the
  previous color in dark mode).
