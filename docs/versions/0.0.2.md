# 0.0.2

**Date:** 2026-04-20.
**Commit:** `6582073`.

Interaction round. Fixed the visible regression from 0.0.1 and filled
in most of the UI surface the prototype was missing.

## Fixes

- **Torus / wireframe axis mismatch**: the shell mesh was rotated by
  π/2 around the X axis, but the node-position function was not.
  Result: nodes floated off the wireframe. Removed the rotation. Both
  now live in the XY plane.

## New interactions

- **Left-click on the torus** picks the nearest node and sets the
  current dyad. OrbitControls' LEFT button is demapped to `-1` so
  orbit never consumes the click.
- **Middle-drag rotates** (remapped from the default left-drag).
- **Right-drag pans.** Scroll still zooms.
- **Reset view** overlay button in the upper-left returns the camera
  and orbit target to the starting framing.
- **Touch**: one-finger orbit, two-finger pan + pinch-zoom.

## Three tracks

- Timeline is now a 2D grid of `tracks × cells`.
- Three default tracks: **Lead** (triangle), **Bass** (sine), **Pad**
  (sawtooth). Each track has an instrument selector (4 waveforms:
  triangle, sine, sawtooth, square) and a mute toggle.
- Audio engine rewrite: `DyadSynth` → `SynthEngine` with one
  `GainNode` bus per track.

## Keyboard

- Arrow keys move a selection cursor in the timeline.
- **Enter** places the current dyad at the selected cell.
- **Backspace** / **Delete** clears it.
- **Space** / **P** outside inputs toggles play.

## First-run welcome tour

- Auto-opens on first visit (`localStorage` flag
  `gom.tour.seen.v1`).
- Covers mouse, keyboard, tracks, resize handle, theme.
- **?** button in the top bar reopens it.

## Theme toggle

- Dark / light theme button on the right of the transport's first
  row. Preference persists in `gom.theme`.
- 3D scene retints via `TorusView.applySceneTheme()`.

## Resizable divider

- Drag the bar between the 3D view and the transport to split upper
  vs lower panel. Double-click to reset. Fraction persists in
  `gom.upper.fraction`.
