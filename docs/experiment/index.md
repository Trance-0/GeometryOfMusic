# Experiments

Each experiment is a specific **musical space**, described on its own
page with three sections:

1. **Music-theory reading** — what this space means if you are a
   musician, and what it makes easy to see or hear.
2. **Mathematical reading** — what this space is as a mathematical
   object, with the identifications and metric that make it one.
3. **How the app renders it** — what is actually on screen, what
   controls drive it, and (for planned experiments) the shape the
   eventual implementation will take.

| Space | Status |
|---|---|

| [Tonnetz (triads)](tonnetz.md) | planned |
| [Dyad orbifold (Möbius)](dyad-orbifold.md) | planned |

| [Voice-leading lattice](voice-leading.md) | planned |
| [Harmonic functions (Schoenberg regions)](schoenberg-regions.md) | planned |

The navbar at the top of the app mirrors this list. Disabled entries
correspond to "planned" experiments — the pages here describe exactly
what each one is meant to render once the implementation lands.
